    二、画线 Bresenham’s line drawing algorithm

    这一章实现了一个线渲染器。
    如何画一条线？
    现在有点 $A(xa,ya)$ 和点 $B(xb,yb)$，要在屏幕上画出线段AB，需要知道除AB之外的在线段上的点。

    • 思路1
      我们假设线段AB 由屏幕上的 $n$ 个点组成，
      那么第$i$个点$P_i$的坐标就是 $(xa + (xb - xa) * (\frac in), yb + (yb - ya) * (\frac in))$
      这也就是插值的思想
      然后我们只要把这 $n$ 个点画出来就行了。
    struct Point {
    int x, y;
    Point operator* (float f) const {
        return {(int) std::round(x * f), (int) std::round(y * f)};
    }
    Point operator+ (const Point& p) const {
        return {x + p.x, y + p.y};
    }
    Point operator- (const Point& p) const {
        return {x - p.x, y - p.y};
    }
};
template <typename T>
T lerp (T a, T b, float t)
{
    return a + (b - a) * t;
}
void line(TGAImage& framebuffer, Point a, Point b, TGAColor color,int n = 100)
{
    for (int i = 0; i < n; i++)
    {
        float t = i / static_cast<float> (n);
        Point P = lerp(a, b, t);
        framebuffer.set(P.x, P.y, color);
    }
}
    • 思路2
      思路1里我们直接假设线段由 $n$ 个点组成，但如果线段的长度特别长呢？我们就会得到一个断断续续的线
      为了解决这个问题，我们换个思路。
      我们遍历 $xa$ 到 $xb$ 的所有 $x$，对于每个 $x$，我们插值算出一个 $y$，从而得到 $P_i$
      但如果 $abs(yb - ya) > abs(xb - xa)$ 时，线段又会变得不连续，所以我们先要判断 $x$ 和 $y$ 哪个跨度更大，然后遍历那一个，得到另一个。
    void line(TGAImage& framebuffer, Point a, Point b, TGAColor color)
{
    bool steep = (std::abs(a.x - b.x) < std::abs(a.y - b.y));
    if (steep)
    {
        std::swap(a.x, a.y);
        std::swap(b.x, b.y);
    }
    if (a.x > b.x)
        std::swap(a, b);
    for (int x = a.x; x <= b.x; x++)
    {
        float t = (x - a.x) / static_cast<float> (b.x - a.x);
        int y = std::round(a.y + (b.y - a.y) * t);
        
        Point P {x, y};
        if (steep)
            framebuffer.set(P.y, P.x, color);
        else
            framebuffer.set(P.x, P.y, color);
    }
}

    现在的写法还有优化空间

    可以优化成下面代码

    void line(TGAImage& framebuffer, Point a, Point b, TGAColor color)
{
    bool steep = (std::abs(a.x - b.x) < std::abs(a.y - b.y));
    if (steep)
    {
        std::swap(a.x, a.y);
        std::swap(b.x, b.y);
    }
    if (a.x > b.x)
        std::swap(a, b);
    float y = a.y;
    float x_len = static_cast<float>(b.x - a.x);
    for (int x = a.x; x <= b.x; x++)
    {
        Point P {x, y};
        if (steep)
            framebuffer.set(P.y, P.x, color);
        else
            framebuffer.set(P.x, P.y, color);

        y += (b.y - a.y) / x_len;
    }
}

    v 开头的是顶点数据，后面的三个数代表这个顶点的 xyz 的值
    vt 开头的是顶点的纹理数据，后面三个数的前两个数代表 uv 的值
    vn 开头的是顶点法线的数据
    f 开头的是三角面的数据，后面由三个顶点组成
    比如 f 6/1/6 1/2/1 2/3/4
    表示这个面由顶点6、1、2组成（每组的第一个代表顶点的索引号，按上面 v 数据的顺序来，从1开始）

    为了更好的处理.obj文件，创建一个Model类来管理
    model.h 定义Model类

    class Model {
    std::vector<vec3> verts;
    std::vector<vec3> triangles;
public:
    Model(const std::string filename);
    const int get_triangles_size() {return triangles.size();}
    std::tuple<const vec3&, const vec3&, const vec3&> get_triangle(int i) 
    {
        return {verts[triangles[i].data[0] - 1], verts[triangles[i].data[1] - 1], verts[triangles[i].data[2] - 1]};
    }
};

    model.cpp 读取.obj文件

    Model::Model(const std::string filename)
{
    std::ifstream file;
    file.open(filename);
    if (file.is_open())
    {
        std::string line;
        while(std::getline(file, line))
        {
            if (line.empty()) continue;
            std::istringstream iss(line);
            char type;
            iss >> type;
            switch (type)
            {
                case 'v':
                {
                    vec3 vert;
                    iss >> vert.x >> vert.y >> vert.z;
                    verts.push_back(vert);
                    break;
                }
                case 'f':
                {
                    vec3 triangle;
                    char dummy_c;
                    int dummy;
                    iss >> triangle.data[0] >> dummy_c >> dummy >> dummy_c >> dummy;
                    iss >> triangle.data[1] >> dummy_c >> dummy >> dummy_c >> dummy;
                    iss >> triangle.data[2] >> dummy_c >> dummy >> dummy_c >> dummy;
                    triangles.push_back(triangle);
                    break;
                }
                default:
                    break;
            }
        }
        file.close();
    }   
}

    main.cpp

    Point project(vec3 v)
{
    return {static_cast<int>((v.x + 1) * width / 2.0), 
        static_cast<int>((v.y + 1) * height / 2.0)};
}
int main(int argc, char** argv) {
    TGAImage framebuffer(width, height, TGAImage::RGB);
    Model model(argv[1]);
    
    std::srand(std::time({}));
    auto start = std::chrono::steady_clock::now();

    for (int i = 0; i < model.get_triangles_size(); i++)
    {
        auto [v1, v2, v3] = model.get_triangle(i);
        line(framebuffer, project(v1), project(v2), red);
        line(framebuffer, project(v1), project(v3), red);
        line(framebuffer, project(v2), project(v3), red);
    }

    auto end = std::chrono::steady_clock::now();
    framebuffer.write_tga_file("framebuffer.tga");
    auto duration = std::chrono::duration_cast<std::chrono::microseconds>(end - start);
    std::cout << std::chrono::duration_cast<std::chrono::milliseconds>(duration).count() << " ms" << std::endl;
    system("start framebuffer.tga");
    return 0;
}


    三、画三角形,光栅化 Triangle rasterization

    • 扫描线光栅化 Scanline rasterization
      已经可以画线框三角形了,现在我们要把三角形填色
      前面给线填色,我们其实是找到对应的点,然后填色
      给三角形填色,则是找到对应的线,然后填色
      一个三角形由三个点组成,我们假设为点$a,b,c$
      找到 $a,b,c$ 中 $y$ 值最高的和最低的 $y_{max},y_{min}$
      我们需要找到 $n$ 条线, $n = y_{max} - y_{min}$ , 即三角形每覆盖的一个 $y$ 值,对应一条线,这条线从三角形当前 $y$ 值的左边到右边
    void triangle(int ax, int ay, int bx, int by, int cx, int cy, TGAImage &framebuffer, TGAColor color) {
    // 按y值排序
    if (ay>by) { std::swap(ax, bx); std::swap(ay, by); }
    if (ay>cy) { std::swap(ax, cx); std::swap(ay, cy); }
    if (by>cy) { std::swap(bx, cx); std::swap(by, cy); }
    int total_height = cy-ay;

    if (ay != by) {
        int segment_height = by - ay;
        for (int y=ay; y<=by; y++) { // 遍历n条线 找到左右边界
            int x1 = ax + ((cx - ax)*(y - ay)) / total_height; 
            int x2 = ax + ((bx - ax)*(y - ay)) / segment_height;
            for (int x=std::min(x1,x2); x<std::max(x1,x2); x++)  // draw a horizontal line
                framebuffer.set(x, y, color);
        }
    }
}

    • 现代光栅化方法
      扫描线的方法,已经可以实现光栅化了
      但只能按顺序执行,所以不能放到GPU上运行
      所以设计了另一种方法,用伪代码可以表示成这样
    triangle(vec2 points[3]) {
    vec2 bbox[2] = find_bounding_box(points);
    for (each pixel in the bounding box) {
        if (inside(points, pixel)) {
            put_pixel(pixel);
        }
    }
}

    每个像素都可以单独着色,所以这部分计算可以交给GPU并行运算
    对于这个方法,我们要实现 find_bounding_box 和 inside 两个函数
    对于 find_bounding_box, 我们可以计算三角形的AABB盒(axis-aligned bounding box),简单来说就是刚好包围三角形的一个正方形,这很好算,只要知道这个正方形的左上角和右下角的坐标就行.
    左上角的x值是三角形三个点x值的最小值,y值是三个点y值的最大值.
    右下角的点同理

    inside 函数需要判断一个点是否在一个三角形内
    有好几种方法
    1.叉积法
    三角形三个点为ABC,需要判定的点为P
    三维向量
    两个向量叉积可以得到垂直与这两个向量的向量,方向符合右手螺旋.
    方向的正负可以来判断两个向量的相对位置(左右)
    所以我们分别让AB叉乘AP,BC叉乘BP,CA叉乘CP,若是得到的三个向量方向相同,那么说明P在三角形ABC内
    2.面积法
    若是点P在三角形ABC内,那么 $S_{ABC} = S_{ABP} + S_{ACP} + S_{BCP}$,因为P点仅仅是把三角形分成了三个区域
    若是点P在三角形ABC外,那么 $S_{ABC} < S_{ABP} + S_{ACP} + S_{BCP}$

    怎么求三角形的面积呢
    二维向量的叉乘是一个标量，其绝对值表示两个向量构成的平行四边形面积，符号表示旋转方向
    三角形面积即为这个平行四边形的一半

    struct Triangle {
    Point a, b, c;
    bool contains(const Point& p) const {
        int w0 = (b.x - a.x) * (p.y - a.y) - (b.y - a.y) * (p.x - a.x);
        int w1 = (c.x - b.x) * (p.y - b.y) - (c.y - b.y) * (p.x - b.x);
        int w2 = (a.x - c.x) * (p.y - c.y) - (a.y - c.y) * (p.x - c.x);
        return (w0 >= 0 && w1 >= 0 && w2 >= 0) || (w0 <= 0 && w1 <= 0 && w2 <= 0);
    }
};

    我们把前面用线渲染器渲染的模型用现在的来填色,把每个三角形随机填色
